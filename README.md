# My Personal Website

## Description
This website was created as part of a project assignment to demonstrate basic HTML structure and webpage organization.

## Technologies Used
- HTML5
- CSS3

## Features
- Feature 1: There are seperate navigation tabs such as About, Projects, and Contact
- Feature 2: When a user hovers over either the about, projects, or contact tab, theres a hover zoom feature on the tab.
- Feature 3: The user is able to enter contact information in the contact tab and submit it.
